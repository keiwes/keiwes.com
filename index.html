Playlister

Playlister is a concept application that would allow multiple Spotify members to listen to the same playlist. Members can suggest songs that are available on Spotify, and a voting system would be in place that would affect the order in which the songs are played. Social features, such as chat and share on social networks, would also be available for a more engaging experience.

My Work

My Work is an employee dashboard designed to increase productivity by providing tools in one location. This dashboard was built in SharePoint as an extension to the My Site functionality. Company web parts and external website web parts were available and customizable. A notification bar was included to alert employees when they needed to perform a work action, or when they received private messages. Brand colors were used.

UX Class

I taught User Experience classes to incoming IT interns. I reviewed the UX field and careers, common web practices, the psychology behind how we use the web, different approaches and tools for wireframing and prototyping, qualitative usability testing, and the company brand. The class was usually about 3 hours in length.
